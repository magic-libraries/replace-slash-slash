### installation

`npm install --save-exact @magic-libraries/replaceSlashSlash`

### usage

in a page/component, just use the lib.replaceSlashSlash function

#### #usage- replaceSlashSlash

```
lib.replaceSlashSlash('//path/into//page//')

// '/path/into/page/'
```
